# Senac Systems Dev

Organização de projetos acadêmicos do curso Técnico em Desenvolvimento de Sistemas — SENAC (Turma 2025/2027).

## Projetos

### UC11 — Modernização da Gestão de Estoque

Sistema com IA para **controle de validade, prevenção de perdas e detecção de anomalias** na rede de Supermercados Atacadão.

[Repositório](https://github.com/senacsystemsdev/UC11) | [Site](https://senacsystemsdev.github.io/UC11/)

**Stack:** React (Vite + TypeScript + Tailwind CSS), Node.js, PostgreSQL 16

```bash
cd mvp/frontend
npm install
npm run dev
```

### Nozesfy — Gestão de Estoque

Plataforma robusta de gestão de estoque e controle para pequenos comércios e empresas, com dashboard analítico, alertas automáticos e integração desktop.

[Repositório](https://github.com/senacsystemsdev/nozesfy) | [Site](http://docs.nozesfy.com)

**Stack:** Next.js 15 (App Router), TypeScript, PostgreSQL (Supabase), Drizzle ORM, Tailwind CSS, JWT, Stripe
